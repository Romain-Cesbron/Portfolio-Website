Incremental Game Website

This project is my take on the "Cookie Clicker" genre of incremental games. In "Cookie Clicker," users initially click on a big cookie on the screen, earning a single cookie per click. They can then use their earned cookies to purchase assets that automatically produce cookies.
I designed my website to look like a Wikipedia page to learn how to replicate a design and layout. I created the website from scratch without directly copying any elements from Wikipedia.
I used JavaScript, HTML, and CSS for this project. I utilized local storage to remember the game state, so even if you close your browser, your game won't start from scratch.
At the end, I connected the project to Firebase to create a working leaderboard, which allowed me to learn a bit about backend development.
